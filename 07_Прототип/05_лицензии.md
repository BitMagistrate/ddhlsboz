## Отношение лицензий

- ЧитАИ — закрытый коммерческий продукт.
- Все open-source компоненты включены в соответствии с их лицензиями.
- Permissive-лицензии (MIT, BSD, Apache, ISC) — допускают коммерческое использование без ограничений (с указанием авторства).
- Copyleft-лицензии (LGPL — psycopg) — изоляция через динамическую линковку, без модификации исходного кода библиотеки.
- При модификации open-source компонентов — изменения публикуются в соответствии с лицензией.

## Реестр публикаций исходников

Если в продакшене будут модификации open-source компонентов с copyleft-лицензиями — изменения публикуются в публичном репозитории и ссылка указывается в подвале сайта.

## Обязательные упоминания

В подвале сайта `chitai.ru` или в разделе «О проекте»:

> Сервис ЧитАИ использует open-source компоненты под лицензиями MIT, BSD, Apache 2.0, PostgreSQL, ISC. Полный реестр и тексты лицензий доступны по ссылке: chitai.ru/licenses.

## Аудит лицензий

- Раз в квартал — автоматическая проверка через `pip-licenses`, `license-checker` (npm).
- При добавлении новой зависимости — проверка лицензии.
- Конфликтные лицензии (GPL, AGPL) — не используются в коммерческом продукте без отдельного согласования.
