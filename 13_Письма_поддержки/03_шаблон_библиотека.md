## Письмо поддержки и о готовности к пилотному сотрудничеству

[Полное наименование библиотеки] (далее — Библиотека) выражает поддержку проекту ООО «ЧитАИ» по созданию российского ИИ-куратора маршрутов чтения и культурного развития для молодёжи 14–22 лет и подтверждает готовность к проведению пилотного проекта.

### О проекте

Проект «ЧитАИ» предлагает Библиотеке:

- виджет на сайт Библиотеки для онлайн-консультаций пользователям;
- интеграция с каталогом Библиотеки (через OPAC) для рекомендаций;
- инструмент аналитики тематических интересов молодёжи в обслуживаемой территории;
- помощь в привлечении молодёжи 14–22 в Библиотеку;
- генератор контента для социальных сетей Библиотеки.

### Готовность Библиотеки

Библиотека выражает готовность:

1. Установить виджет ЧитАИ на сайте Библиотеки на пилотный период 6 месяцев.
2. Подключить каталог Библиотеки к платформе для рекомендаций пользователям.
3. Обучить 3–5 библиотекарей работе с инструментом.
4. Регулярно предоставлять обратную связь о работе сервиса.
5. Содействовать сбору обезличенной аналитики использования сервиса.
6. Согласовывать публикации совместных кейсов после завершения пилота.

### Условия пилота

- Пилот проводится на безвозмездной основе.
- Соблюдение 152-ФЗ — обязательно.
- Хранение и обработка ПД пользователей — на инфраструктуре ЧитАИ в РФ.
- Контроль над виджетом — по решению Библиотеки.
- Право Библиотеки прекратить пилот в любой момент.
- Совместные мероприятия (выставки, лекции) — по согласованию.

### Ожидания Библиотеки

- Рост посещаемости Библиотеки молодёжью 14–22 лет на 15–20 п.п.
- Расширение охвата онлайн-аудитории.
- Аналитический инструмент для понимания тематических интересов молодёжи.
- Поддержка задач Программы национального чтения и нацпроекта «Культура».

## Внутренние комментарии

- Шаблон рассчитан на разовое подписание после первой встречи.
- Перед обращением:
  1. Демонстрация продукта директору.
  2. Согласование с заместителем по информационным технологиям.
  3. Согласование с пресс-службой.
  4. Технический аудит виджета (как он встроится).
  5. Подписание письма.

- На момент подачи в Нейрофест допустимо: «Ведём переговоры с N библиотеками. Письма поддержки в процессе оформления».

- Какие библиотеки выбирать (приоритет):
  - Региональные универсальные научные библиотеки.
  - Центральные муниципальные библиотеки крупных городов.
  - «Модельные библиотеки» (программа «Модельные библиотеки» Минкульта).
  - Библиотеки в регионах с активной работой по нацпроекту «Культура».
  - Библиотеки с активным online-присутствием.

- Аналогичный шаблон для музея — с заменой:
  - «Библиотека» → «Музей».
  - «Каталог» → «Экспозиция».
  - «Виджет» → «Виджет / QR-метки в экспозиции».
  - «Библиотекари» → «Кураторы / экскурсоводы».
  - «Посещаемость библиотеки» → «Посещаемость музея».

## Дополнительные обязательства ЧитАИ перед партнёрами (общие для всех типов)

- Прозрачность всех аналитических метрик.
- Открытость технологического стека (на сайте — список используемых моделей).
- Готовность к аудиту по 152-ФЗ.
- Отсутствие коммерческих условий в пилотный период.
- Готовность к совместным публикациям после завершения пилота (с согласия партнёра).
- Возможность партнёра прекратить сотрудничество в любой момент без обязательств.
