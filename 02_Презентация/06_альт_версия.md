# Альтернативная (короткая) версия презентации ЧитАИ — 6 слайдов

Для коротких форматов: 3-минутный доклад, рассылка инвестору, питч в Минкульте.

## Слайд 1 — Что это

```
ЧитАИ — российский ИИ-куратор маршрутов чтения
по фондам РГБ и НЭБ.
Для молодёжи 14–22, учителей и регионов.
```

## Слайд 2 — Проблема и решение в 3 строки

```
9,2 млн «Пушкинских карт». <2% операций — на библиотеки.
Учитель тратит 6–8 часов в неделю на материалы.
ЧитАИ собирает маршрут «5 шагов на 4 недели»
со ссылками на оригиналы из РГБ/НЭБ.
```

## Слайд 3 — Технология

```
RAG (поиск + ответ) на YandexGPT 5 Pro и GigaChat MAX.
Иллюстрации — Kandinsky 3.1, озвучка — SpeechKit.
PostgreSQL + pgvector. Yandex Cloud.
Соответствие 152-ФЗ и Указу №490.
```

## Слайд 4 — Демо и эффект

```
Куратор · Тренажёр ЕГЭ · Дашборд региона.
+12 п.п. вовлечённости молодёжи в регионе.
-4 ч/нед. учителю на сбор материалов.
KPI нацпроекта «Культура».
```

## Слайд 5 — Бизнес-модель

```
7 каналов: «Пушкинская карта», B2G-регионы,
гранты, white label, ЕГЭ-тренажёр, аналитика, B2B-сети.
Гипотеза 270 млн ₽ ёмкости на 24 мес.
ARPU B2C 3 480 ₽/год, payback B2C 3–4 мес.
```

## Слайд 6 — Команда, план, контакты

```
Команда: CEO · CTO · Backend · Frontend · Designer · Куратор.
План: M2 MVP → M4 пилот → M6 регион → M12 6 000 B2C →
M18 60 000 B2C / 6 регионов.
Контакт: ЧитАИ · chitai.ru · @ChitaiBot · [имя]
```

## Заметки спикера для 3 минут

- Слайд 1 — 15 сек: представить проект, аудиторию, технологию.
- Слайд 2 — 30 сек: цифры проблемы и формулировка решения.
- Слайд 3 — 30 сек: коротко стек и почему российский.
- Слайд 4 — 30 сек: демо плюс эффект.
- Слайд 5 — 35 сек: бизнес-модель, ёмкость и юнит-экономика.
- Слайд 6 — 40 сек: команда, план, призыв.

Всего 180 секунд.
