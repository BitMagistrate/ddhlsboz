## Слайд 2 — Проблема

```
9,2 млн молодых людей
держат в руках «Пушкинскую карту».
Меньше 2% операций — на библиотеки.
Учитель тратит 6–8 часов в неделю на сбор
материалов к уроку.
```

## Слайд 3 — Кому больно

```
Школьник 14–22 — нет персонального куратора.
Учитель литературы и истории — теряет часы.
Региональная библиотека — теряет молодёжь.
Региональное министерство — не видит, что читают.
```

## Слайд 4 — Решение

```
1. Запрос: «Хочу разобраться в Серебряном веке».
2. ЧитАИ строит маршрут на 4 недели — книги, лекции,
   аудио, музеи, события «Пушкинской карты».
3. Учитель получает сценарий урока,
   регион — дашборд активности.
```

## Слайд 5 — Технология

```
RAG: гибридный поиск + ответ российской LLM.
Каждый ответ — со ссылкой на источник.
Если источника нет — модель отвечает «нет данных».
Цель: «галлюцинации» ≤ 1%.
```
