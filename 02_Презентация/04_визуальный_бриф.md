# Визуальный бриф презентации ЧитАИ (Kandinsky 3.1 + дизайн-система)

## Общие принципы

- Стиль: спокойная российская академическая эстетика, отсылки к библиотечной культуре и архивам, без «техно-неонового» 2010-х.
- Цветовая палитра базовая: глубокий синий #0E2A47 (РГБ-стиль), тёплый бежевый #EFE3CC (бумага), акцент терракотовый #B0573B (закладка), нейтральный графитовый #1F2937, белый #FFFFFF.
- Шрифт заголовков: PT Serif (русский). Шрифт текста: Inter / PT Sans.
- Сетка: 12 колонок, 16:9, отступ 64 пикселя.
- Иллюстрации: 2–3 ключевых стиля (читальный зал, силуэты разных аудиторий, абстрактная схема RAG в плоской графике).

## Бриф иллюстраций для Kandinsky 3.1

### И1. Обложка (слайд 1)

```
Стиль: спокойная плоская иллюстрация, мягкий академический модерн.
Композиция: фрагмент читального зала классической библиотеки, перед глазами — открытая книга, на странице которой проступает тонкий светящийся узор сетевого графа (символ ИИ).
Цвета: тёплые бежевые и глубокий синий. Терракотовый акцент на корешке книги.
Без людей. Без логотипов сторонних брендов.
Без надписей на иллюстрации.
Соотношение: 16:9, разрешение 1920×1080.
Не использовать: лучи, неон, сияние, «кислотные» цвета, западные стилизации.
```

### И2. Слайд «Кому больно»

```
Стиль: четыре силуэта-стилизации в один ряд: школьник у учебника, учитель у доски, библиотекарь среди стеллажей, чиновник за рабочим столом с экраном дашборда. Минимализм, без лиц (силуэты).
Палитра: бежевые, синие, терракотовые акценты.
Атмосфера: спокойная, рабочая, без драматизации.
Соотношение: 16:9.
Не использовать: эмодзи, мультяшный стиль, гиперреализм, западные интерьеры.
```

### И3. Слайд «Технология»

```
Стиль: плоская инфографика — упрощённая схема RAG.
Элементы: блок «Запрос», блок «Поиск (BM25 + вектор)», блок «Reranker», блок «Synthesizer (YandexGPT/GigaChat)», блок «Ответ + источники».
Без названий брендов на иллюстрации (только в подписях слайда).
Палитра: основная (см. выше).
Соотношение: 16:9.
Не использовать: 3D-эффекты, лучевой стиль, излишне сложную графику.
```

### И4. Слайд «Демо»

```
Имитация скриншотов трёх экранов: «Куратор», «Тренажёр ЕГЭ», «Дашборд региона». Можно изобразить как стилизованные карточки. Тексты и цифры — собственные.
Соотношение: 16:9.
```

### И5. Слайд «Эффект»

```
Стиль: карточки с инфографикой и метриками. Без иллюстрации в фоне; только цифры и иконки в линейном минимализме.
Иконки: библиотека, школа, музей, регион (министерство).
Соотношение: 16:9.
```

### И6. Слайд «Просьба»

```
Стиль: лаконичная композиция: открытая книга на пюпитре с закладкой-стрелкой, как символ продолжения. Палитра — основная.
Без людей.
Соотношение: 16:9.
```

## Иконография

- Иконки в линейном стиле, толщина обводки 1,5 px, без заливки в основном состоянии.
- Источник: собственная библиотека, при отсутствии — Lucide или Phosphor (с переработкой стилистики).
- Не использовать: цветные эмодзи, иконки Material Design в чистом виде, «мультяшные» иконки.

## Чек-лист отказа

- Без эмодзи в верстке.
- Без слов: «инновационный», «уникальный», «революционный», «прорывной».
- Без слова «погружение» в иллюстрациях и подписях.
- Без западных стилистических отсылок (статуя Свободы, Большой Бен и пр.).
- Без религиозной символики в иллюстрациях, кроме случаев, когда это необходимо для исторической темы (тогда — корректно и с консультацией).
