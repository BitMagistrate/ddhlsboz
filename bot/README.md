# ЧитАИ — Telegram-бот (демо)

Aiogram 3.x, ходит в backend по тем же эндпоинтам, что и веб-фронтенд.

## Что умеет

- `/start` — приветствие и меню (Куратор / Тренажёр / Пушкинская карта / О проекте)
- Свободный текст (например, «Хочу понять Пушкина за 4 недели») — возвращает 4-недельный маршрут чтения с цитатами и ссылками на public domain.
- Кнопка «Тренажёр ЕГЭ» — выдаёт демо-вопрос по литературе.
- Кнопка «Пушкинская карта» — пояснение демо-логики (в продакшене подмешиваются события партнёров).

## Запуск

```bash
pip install -r requirements.txt
export BOT_TOKEN=...                       # у @BotFather
export API_BASE=https://chitai-api.fly.dev # или http://127.0.0.1:8000
python bot.py
```

## Что заменяется в проде

- `API_BASE/api/curator/route` — сейчас mock-RAG поверх public-domain корпуса. В проде — pgvector-индекс фондов РГБ/НЭБ + YandexGPT 5 Pro / GigaChat MAX.
- Источники Пушкинской карты — официальный каталог через API.
- Логирование диалога — Yandex Cloud Logging, хранение в РФ (152-ФЗ).
- Антипромпт-инжекция и фильтр галлюцинаций — слой над генерацией.
