# Методичка для учителя литературы

## Назначение

Методичка описывает, как учитель литературы 5–11 класса использует ЧитАИ для подготовки уроков, домашних заданий и подготовки учеников к ОГЭ и ЕГЭ. Согласована с Федеральным государственным образовательным стандартом (ФГОС ООО, ФГОС СОО) и материалами ФИПИ.

## 1. Что умеет ЧитАИ для учителя

- Сценарий урока на 45 минут по конкретной теме (планируемые результаты, ход урока, домашнее задание).
- Карточки заданий по структуре ФИПИ (анализ текста, аргументация, сочинение).
- Подборка иллюстраций, аудио и видео по теме.
- Маршрут чтения для класса на 4 недели по тематическому блоку.
- Автоматическая проверка письменных работ с обратной связью.
- Аналитика по классу: тематические интересы, прогресс, слабые места.

## 2. Сценарий первого использования

### Шаг 1. Регистрация

1. Откройте `chitai.ru`.
2. Зайдите как «Учитель» через e-mail школы.
3. Подтвердите согласие на обработку ПД.
4. Заполните анкету: предмет, класс, школа.

### Шаг 2. Создание класса

1. В кабинете нажмите «Создать класс».
2. Введите название (например, «10А, литература, 2026»).
3. Получите код подключения для учеников.

### Шаг 3. Подготовка первого урока

1. В чате с ЧитАИ-куратором введите запрос:
   > «Сценарий урока для 10 класса по теме „Преступление и наказание", глава „Сон Раскольникова". Уровень — углублённый. ФГОС ООО.»
2. ЧитАИ выдаст:
   - планируемые результаты (предметные, метапредметные, личностные);
   - ход урока с таймингом;
   - перечень источников;
   - задания для домашней работы;
   - вопросы для дискуссии.
3. Скачайте сценарий в DOCX или PDF, отредактируйте при необходимости.

### Шаг 4. Назначение домашнего задания

1. В кабинете класса перейдите в «Задания».
2. Выберите готовое задание ФИПИ или создайте на основе сценария.
3. Установите срок сдачи.
4. Ученики получат задание в личном кабинете и в Telegram-боте.

### Шаг 5. Проверка работ

1. По истечении срока система собирает работы.
2. ЧитАИ предоставляет первичную проверку (грамотность, наличие аргументов, соответствие критериям ФИПИ).
3. Учитель просматривает, корректирует, выставляет оценку.

## 3. Соответствие ФГОС

### ФГОС ООО (5–9 классы)

ЧитАИ помогает достичь:

- предметных результатов: знание содержания произведений, понимание идейно-тематического содержания, умение анализировать;
- метапредметных результатов: умение работать с информацией, аргументировать, строить тексты;
- личностных результатов: формирование российской гражданской идентичности, ценностного отношения к культуре.

### ФГОС СОО (10–11 классы)

ЧитАИ работает с углублённым уровнем:

- историко-литературный процесс;
- стилевые особенности эпох;
- интертекстуальные связи;
- подготовка к ЕГЭ и итоговому сочинению.

## 4. Критерии ФИПИ для оценки сочинений

Сценарий урока по сочинению включает критерии ФИПИ:

1. К1. Формулировка проблемы.
2. К2. Комментарий к проблеме (с примерами).
3. К3. Отражение позиции автора.
4. К4. Отношение пишущего к позиции автора.
5. К5. Смысловая цельность, речевая связность.
6. К6. Точность и выразительность речи.
7. К7–К12. Грамотность.

ЧитАИ при анализе сочинения отмечает каждый критерий и даёт обратную связь.

## 6. Типы заданий

- Анализ текста (5–11).
- Сравнительный анализ двух произведений.
- Сочинение по проблеме.
- Аргументированный ответ на вопрос.
- Подбор цитат к тезису.
- Подготовка устного выступления.
- Создание интеллект-карты по произведению.

## 7. Аналитика по классу

В кабинете отображаются:

- средний прогресс по темам;
- топ-5 интересных учеников по активности;
- слабые места класса (темы, по которым низкие баллы);
- рекомендации по корректировке тематического планирования.

## 8. Этический протокол для учителя

Учитель обязуется:

- использовать сценарии и задания как методическую основу, а не как готовый ответ;
- проверять работы учеников лично, ИИ — только в качестве помощника;
- не передавать пароли и доступы третьим лицам;
- информировать родителей об использовании сервиса в учебном процессе.

## 9. Часто задаваемые вопросы

**1. Заменит ли ЧитАИ учителя?**
Нет. ЧитАИ — инструмент подготовки и аналитики. Решения принимает учитель.

**2. Как защищены данные учеников?**
В соответствии с 152-ФЗ, локализация в РФ, шифрование, обезличивание в выгрузках.

**3. Что делать, если ЧитАИ дал неточный ответ?**
Сообщите через кнопку «обратная связь». Каждое сообщение разбирается командой.

**4. Можно ли работать без интернета?**
В частичном режиме — сценарий можно скачать и работать офлайн. Чат и тренажёр требуют подключения.

**5. Сколько стоит для школы?**
До 10 учителей в одной школе — бесплатно в пилотный период. Тариф для образовательных организаций — по запросу.
