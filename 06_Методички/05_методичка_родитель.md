# Методичка для родителя

## Назначение

Методичка описывает, как родитель школьника 14–18 лет использует ЧитАИ совместно с ребёнком для подготовки к ОГЭ/ЕГЭ, для чтения по школьной программе и для культурного развития.

## 1. Что умеет ЧитАИ для семьи

- Маршрут чтения по школьной программе литературы и истории.
- Тренажёр ЕГЭ/ОГЭ с автоматической проверкой.
- Афиша культурных событий для всей семьи (с «Пушкинской картой» — для ребёнка).
- Совместное чтение: маршрут для родителя и ребёнка.
- Прогресс-отчёт по подготовке к экзамену.

## 2. Регистрация

### Шаг 1. Регистрация родителя

1. Откройте `chitai.ru` или установите приложение.
2. Зайдите как «Родитель» через e-mail.
3. Подтвердите согласие на обработку ПД.
4. Заполните анкету: ФИО, e-mail, ребёнок (имя, возраст, класс).

### Шаг 2. Привязка ребёнка

1. В кабинете нажмите «Добавить ребёнка».
2. Подтвердите законное представительство (галочка о согласии).
3. Получите код для передачи ребёнку.
4. Ребёнок входит со своего устройства, вводит код, ему предоставляется доступ в режиме «14+».

### Шаг 3. Совместное использование

В семейном кабинете:

- родитель видит прогресс ребёнка по школьной программе и тренажёру ЕГЭ;
- родитель и ребёнок могут вести совместный список «к прочтению»;
- родитель получает уведомления о ключевых результатах ребёнка (по согласованию с ребёнком).

## 3. Подготовка к ЕГЭ и ОГЭ

ЧитАИ помогает:

- спланировать маршрут подготовки на 6–12 месяцев;
- разобрать каждый тип задания ФИПИ;
- получать обратную связь по сочинениям;
- видеть прогресс по баллам;
- сравнить динамику с типичной для класса.

## 4. Возрастной режим 14+

В режиме 14+ применяются дополнительные фильтры:

- исключение шок-контента, насилия, тематики самоповреждения;
- материалы по чувствительным темам — с дисклеймером и рекомендацией обсудить с родителем или педагогом;
- упоминание фактов истории и литературы — нейтральное, без эмоциональной окраски.

## 5. «Пушкинская карта» (для ребёнка 14–22)

ЧитАИ помогает:

- найти музей, театр, концерт по интересам ребёнка;
- забронировать или купить билет (через партнёрский сервис);
- спланировать культурный маршрут на месяц.

## 6. Совместное чтение

Маршрут «совместное чтение» — для родителя и ребёнка вместе. Например:

- «Война и мир» Толстого — 4 недели, по тому в неделю;
- цикл рассказов А.П. Чехова — 2 недели;
- русская военная проза — 6 произведений на лето.

После каждого этапа — короткие вопросы для обсуждения.

## 7. Соответствие 152-ФЗ

- Регистрация ребёнка — с согласия законного представителя.
- Все данные хранятся в РФ.
- Шифрование, ограниченный доступ.
- Права родителя: запросить данные ребёнка, удалить аккаунт, отозвать согласие.

## 8. Часто задаваемые вопросы

**1. С какого возраста можно использовать сервис?**
С 14 лет — самостоятельно (с согласием законного представителя в школьном кабинете). С 18 — без ограничений.

**2. Стоит ли это денег?**
Базовые функции бесплатны. Тренажёр ЕГЭ/ОГЭ — платная подписка от 290 ₽/мес. Подробнее — на странице тарифов.

**3. Будет ли ЧитАИ заменой репетитора?**
Нет. ЧитАИ — инструмент подготовки. Если ребёнок нуждается в индивидуальной работе с педагогом, ЧитАИ дополняет, но не заменяет.

**4. Как ребёнок не отвлекается на посторонние темы?**
Возрастной режим 14+ исключает шок-контент. Сервис фокусируется на литературе, истории, культуре. Чат не имеет «свободного» режима для не предметных разговоров.

**5. Будет ли мой ребёнок в безопасности?**
Все коммуникации — внутри сервиса. Сервис не публикует переписки, не делится данными ребёнка с третьими лицами.

**6. Можно ли подключить нескольких детей?**
Да. В семейном кабинете до 5 детей.
