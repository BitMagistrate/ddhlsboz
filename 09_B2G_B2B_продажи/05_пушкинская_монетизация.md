## 1. Что такое Пушкинская карта в 2025–2026 (только факты)

| Параметр | Значение |
|---|---|
| Целевая аудитория | граждане РФ **14–22 года включительно** |
| Лимит | **5 000 ₽/год** (1 января пополнение, 31 декабря — сгорание) |
| Сублимит на кино | **до 2 000 ₽** только на отечественные фильмы из списка |
| Карта | виртуальная или пластиковая «Мир» |
| Что **нельзя** | снимать наличные, переводить, частичная оплата, покупать товары/услуги вне утверждённой афиши, передавать другому человеку |
| Что **можно** | покупать **именные билеты** на культурные мероприятия из афиши Минкульта (Культура.РФ / Госуслуги.Культура) |
| Афиша содержит | спектакли, концерты, выставки, экскурсии, **мастер-классы**, **обучающие/просветительские лекции**, **интеллектуальные квизы и игры**, фестивали, иммерсивные проекты, кинопоказы отечественного кино |
| Банк-оператор | до 31.12.2025 — **Почта Банк**, с 01.01.2026 — **ВТБ** (BIN: 22002482, 22002483, 22002484) |
| Платформа подключения организаций | **PRO.Культура.РФ** (pro.culture.ru) |
| Эквайринг | «**белый**» терминал, привязанный в личном кабинете PRO.Культура к юр.лицу/ИП, согласованный с банком-оператором |
| Активных пользователей | ~6–8 млн молодых россиян (по разным оценкам Минкульта) |

### Ключевые ограничения, которые меняют продуктовую логику

1. **ПК — это билет, а не товар.** Купить «подписку Premium» напрямую с ПК нельзя. Можно продать **билет на мероприятие** (онлайн-лекцию, квиз, мастер-класс), а уже доступ к AI-инструментам выдавать как «раздаточные материалы лектория».
2. **Только культурно-просветительский характер.** Чистый «решу ЕГЭ по математике» Минкульт обычно режет. А «Лекторий по русской литературе», «Курс истории искусств», «Биология как культурный феномен», «Русский язык: от Пушкина до Бродского» — проходят, и это документированный кейс библиотек/университетов.
3. **Билет именной + паспорт.** На входе на офлайн-мероприятие проверяют. Для онлайн — упрощённо, но фамилия в билете должна совпадать с владельцем карты.
4. **Полная сумма только с ПК.** Нельзя «1500₽ с ПК + 200₽ доплаты». Цена билета должна полностью укладываться в остаток по карте, иначе оплата не пройдёт.
5. **Деньги сгорают 31 декабря** → у программы мощный сезонный спайк в **октябре–декабре**.

## 2. Что у нас может стать «билетом по Пушкинской карте»

Соотносим функционал нашего сервиса с тем, что Минкульт реально утверждает:

| Наш продукт / фича | Как упаковать как «мероприятие по ПК» | Шанс модерации |
|---|---|:-:|
| Live-лекция с AI-конспектом и Q&A | «Просветительская лекция» на гуманитарную/культурологическую тему | **высокий** ✓ |
| Quiz Royale (наш мульти-квиз) | «Интеллектуальная игра» / «Литературный вызов» | **высокий** ✓ (десятки прецедентов в библиотеках) |
| Knowledge Climb / мини-игры | Сложно — слишком развлекательно. Сработает только в составе **фестиваля/мастер-класса** | средний |
| Подкаст из материалов (Audio Recap) | «Аудиолекторий» — серия онлайн-аудиолекций | средний-высокий |
| Tutor Me (AI-урок) на культурные темы | «Курс из N просветительских занятий» | средний-высокий |
| Иммерсивный AI-чат с «Пушкиным/Достоевским» | «Иммерсивный литературный проект» | **высокий** ✓ (документированный формат, есть вебинары Минкульта об этом) |
| Mind Map по теме | как доп.материал внутри лекции | — |
| Essay Grader / помощь с эссе | «Мастер-класс по академическому письму» | средний |
| AI-конспекты по школьному учебнику математики | вне формата ПК | низкий ✗ |
| AI-курсовые / решение задач (zachet.tech) | вне формата ПК (это услуга, а не культурное мероприятие) | низкий ✗ |
| Электронные книги / sw-подписка | вне формата ПК (это товар/подписка, не билет) | низкий ✗ |

**Вывод:** ставку делаем на **четыре формата**:
1. Онлайн- и офлайн-**лектории** на гуманитарные темы.
2. **Интеллектуальные игры** (Quiz Royale в просветительской обёртке).
3. **Иммерсивные онлайн-проекты** (AI-чат с классиком, AI-экскурсия по картине).
4. **Мастер-классы** (по эссе, по подготовке доклада, по финансовой грамотности с культурным контекстом и т.д.).

## 3. Юридическая сборка (что нужно сделать ОДИН раз)

### 3.1 Юр.лицо и ОКВЭД
ИП или ООО подойдёт, важны коды:
- **90.04** — деятельность учреждений культуры и искусства (главный)
- **91.01** — деятельность библиотек и архивов (если делаете партнёрство-обёртку)
- **85.41** — образование дополнительное детей и взрослых
- **90.01** — деятельность в области исполнительских искусств
- **63.91** — деятельность информационных агентств (для онлайн-контента)

### 3.2 Регистрация на PRO.Культура.РФ
1. Заводите кабинет организации на **pro.culture.ru**.
2. Подаёте заявление в **региональный экспертный совет** Министерства культуры (письмо за подписью руководителя на имя министра культуры субъекта).
3. Совет рассматривает 2–6 недель и одобряет/отказывает по каждому **мероприятию отдельно**.
4. Каждое мероприятие проходит модерацию Минкульта (контент, репутация ведущих, культ.значимость).

### 3.3 «Белый» терминал и эквайринг
- Получаете POS-терминал от банка-эквайера (Сбер, ВТБ, Тинькофф, Альфа — любой).
- В кабинете PRO.Культура регистрируете терминал как «белый», добавляете BIN банка-оператора (с 2026 — ВТБ: 22002482, 22002483, 22002484).
- Акцептуете оферту ВТБ для оплаты билетов с ПК.
- Для онлайн-продаж используете **билетные агрегаторы**, уже интегрированные с программой: **UBSystem (ub-system.ru)**, Яндекс.Афиша, Kassir.ru, Quick Tickets, Рамблер/Касса. Они снимают головную боль интеграции и берут 4–10% комиссии.

### 3.4 Касса по 54-ФЗ
Облачные кассы под онлайн-эвенты: **АТОЛ Онлайн**, **Эвотор**, **Modulkassa**, **CloudPayments + ОФД**. Чек уходит покупателю на e-mail / в приложение.

### 3.5 Что не нужно
- Не нужно лицензии на образование, **если вы продаёте «культурно-просветительские мероприятия»**, а не образовательные услуги с выдачей государственного документа об образовании.
- Сертификаты слушателя курса оформляются как «удостоверения о прохождении лектория» без лицензии Рособрнадзора.

## 4. 8 конкретных способов заработать на ПК внутри нашего сервиса

Цена ниже — ориентир. Все цифры — **базовые предположения**, надо тестировать.

### Способ 1. Подписка-эквивалент через **«Просветительский абонемент»**
- Продаём билет «**Абонемент на 12 ежемесячных онлайн-лекций по русской литературе**» — **4 990 ₽**.
- Это съедает 99,8% годового лимита ПК.
- Доступ к AI-инструментам (конспекты, флешкарты, мини-игры) даём как «раздаточные материалы лектория» — bundled.
- По сути это наш Premium-план, оплаченный государством.

**Что нужно по продукту:** регулярный календарь лекций, спикеры (можно своими силами + приглашённые), Telegram-канал или приложение для трансляций, после-лекционные материалы.

### Способ 2. **Турниры Quiz Royale на ПК**
- Серия еженедельных онлайн-квизов «Литературный вызов» / «Исторический турнир» / «Кинокультурология».
- Билет участника — **300–500 ₽**.
- Призовой фонд — символический (книги, мерч), главная мотивация — лидерборд + сертификат.
- Это идеально вписывается в нашу мини-игру **Quiz Royale**: один и тот же движок, контент пишет AI.
- **Виральность:** PIN-код, друзья присоединяются за билет, отчёты в TikTok.

**Юнит-экономика:** 1 квиз × 200 участников × 400 ₽ = **80 000 ₽** за вечер. Вычитаем комиссии (~10%) и налоги (УСН 6%) — ≈ 67 000 ₽ чистыми. Минимальные операционные расходы (1 ведущий + auto-content).

### Способ 3. **Иммерсивный AI-проект «Чат с классиком»**
- Билет на 60-минутную онлайн-сессию: пользователь общается голосом/текстом с AI-агентом, стилизованным под Пушкина / Достоевского / Цветаеву / Бродского. Контекст — наш RAG по их произведениям и биографиям.
- Цена — **600–900 ₽**.
- Технически это **наш AI-Tutor + Voice Call** + специально сделанный character system prompt + RAG по литературоведческим источникам.
- Категория «литературный иммерсивный проект» уже одобряется Минкультом (есть прецеденты вебинаров pro.culture.ru на эту тему в 2024).

### Способ 4. **Кинолекторий: показ + AI-разбор**
- Партнёримся с прокатчиком отечественного кино (по афише ПК) или с региональным киноклубом.
- Продаём «билет на кинолекторий» — лекция + просмотр фильма (отечественного) — **800–1 200 ₽** (часть в кино-сублимит, часть в основной).
- Внутри лекции: интерактивный AI-разбор (наш Spark.E Visuals по кадрам), мини-квиз про режиссёра.
- Студент получает в нашем сервисе Audio Recap фильма + конспект + флешкарты по эпохе.

### Способ 5. **B2B school/college: класс под ключ**
- Договор со школой/колледжем: проводим в их библиотеке/актовом зале серию интеллектуальных игр на наших технологиях.
- Школа собирает группы 30–100 человек, продаёт билеты учащимся **по 200–400 ₽** через PRO.Культура.
- Делим выручку 60/40 или 70/30.
- Это самый «жирный» канал: одна школа = 500 учеников × 4 события в год × 300 ₽ = **600 000 ₽/год**.
- 50 школ-партнёров = **30 млн ₽/год** только B2B-школьный канал.

### Способ 6. **Мастер-классы «Как написать эссе/доклад»**
- Билет 700–1 000 ₽, 90 минут онлайн.
- Используем наш **Essay Grader**: каждый участник присылает свой текст, AI разбирает с фидбэком, лектор комментирует топ-5 кейсов.
- Темы: «Эссе по русской литературе», «Доклад по истории», «Сочинение для итогового», «Как написать research paper».

### Способ 7. **Аудиолекторий-подкаст** на основе Audio Recap
- Билет на «онлайн-аудиолекцию» — 200–300 ₽.
- В подписке — 1 эпизод/неделя по гуманитарным темам, генерится автоматически нашим AI (NotebookLM-style диалог двух ведущих).
- Студент получает .mp3 + конспект + квиз для самопроверки.
- При низкой стоимости и автоматическом производстве это **высокомаржинальный** «long-tail» — масштабируется без операционных расходов.

### Способ 8. **Партнёрство с библиотеками и музеями (white-label)**
- Уже одобренные участники программы (региональные библиотеки, музеи, ДК) часто **не имеют технологий** для проведения интерактивных мероприятий.
- Даём им white-label-инструмент: «загрузите PDF/коллекцию → получите готовую интеллектуальную игру / квест / иммерсивный AI-чат / лекторий».
- Они продают билеты по своей PRO.Культура-аккредитации, мы получаем **30–40% от GMV** или фикс. SaaS-абонемент 30–80 тыс ₽/мес.
- В РФ ~12 000 библиотек и ~3 500 музеев, многие активно ищут контент. Это **самый защищённый канал** — наш контракт с ними не зависит от того, одобрят ли наши собственные мероприятия.

## 5. Прикидка выручки (нижняя оценка)

| Канал | Базовый сценарий | Месячная выручка |
|---|---|---:|
| Просветительский абонемент (4990 ₽/год) | 1 000 продаж/мес в пиковый Q4, 200/мес в среднем | ~1 200 000 ₽ |
| Quiz Royale-турниры | 4 турнира/неделю × 150 участников × 400 ₽ | ~960 000 ₽ |
| Иммерсивный «Чат с классиком» | 200 сессий/мес × 700 ₽ | ~140 000 ₽ |
| Кинолектории (партнёрство) | 8 событий/мес × 150 человек × 1000 ₽ × 35% наша доля | ~420 000 ₽ |
| B2B-школы | 20 школ × 4 события/год × 70 чел × 300 ₽ × 65% наша доля | ~227 000 ₽/мес |
| Мастер-классы по эссе | 12/мес × 80 чел × 800 ₽ | ~768 000 ₽ |
| Аудиолекторий | 3 000 продаж × 250 ₽ | ~750 000 ₽ |
| White-label для библиотек/музеев | 30 партнёров × 50 000 ₽ SaaS | ~1 500 000 ₽ |
| **Итого нижняя оценка** | | **~5,9 млн ₽/мес** |

В **пиковый Q4** (ноябрь–декабрь, когда деньги на ПК сгорают) умножайте на ~2,5–3.

С 50 школами и 100 white-label партнёрами модель легко масштабируется в **20–40 млн ₽/мес** ARR-эквивалента. И всё это **поверх** обычной подписочной модели сервиса.

## 7. Риски и как их закрыть

| Риск | Что делает | Как закрываем |
|---|---|---|
| Минкульт отклонит мероприятие («не культурно-просветительское») | теряете 2–6 недель и репутацию | каждое наше мероприятие пишем строго в формате «лекторий по гуманитарной теме», избегаем формулировок про ЕГЭ/«натаскивание» |
| Отзыв аккредитации за нарушения | блокировка терминала, штрафы | строго именные билеты, сверка паспортов на входе, ведём журнал учёта |
| Сезонность (всё съедают в декабре) | кэш-флоу неровный | бюджет планируем на квартал; в Q1 бесплатный/freemium-канал, чтобы компенсировать |
| Смена банка-оператора (2025→2026 ВТБ) | старые BIN перестали работать | в 2025 г. переподключить терминалы с BIN 22002482/3/4, акцептовать оферту ВТБ |
| Возвраты и фрод (купил → отменил → купил у конкурента) | минус выручка, минус репутация | билет невозвратный за 24ч до события (по правилам ПК — допустимо); fingerprint-защита |
| Имиджевый риск: «обходим ЕГЭ через ПК» | публичная критика, отзыв аккредитации | публичный позиционинг — «культурное просвещение», не «инструмент сдать экзамен» |
| Юр.риск 152-ФЗ (паспорта на входе) | штрафы Роскомнадзора | данные не храним, только сверяем; согласие на обработку — в момент покупки билета |

## 8. Что я бы сделал по шагам

1. **Регистрация юр.лица и ОКВЭД** (1–2 недели) — ИП / ООО, культурные коды.
2. **Профиль на PRO.Культура.РФ + первое заявление в региональный экспертный совет** (3–6 недель ожидания).
3. **Параллельно** — интеграция с **UBSystem** или собственная касса под 54-ФЗ + «белый» терминал ВТБ (2–3 недели).
4. **Запуск первого пилота: серия из 4 онлайн-лекций «Литература в эпоху AI»** — даже если 100 билетов по 700 ₽ — это уже валидирует все процессы.
5. **Подключение Quiz Royale как «интеллектуальной игры»** — первый раз требует отдельного одобрения, дальше идёт пачкой.
