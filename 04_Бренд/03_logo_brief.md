# Бриф логотипа ЧитАИ

## Назначение

Логотип используется в:

- Шапке сайта `chitai.ru`.
- Аватаре бота `@ChitaiBot`.
- Презентациях, документах, юр. бланках.
- Печати на фирменных носителях (бейдж, наклейка, визитка).
- Видеороликах.
- Социальных сетях.

## Концепция

Знак сочетает:

1. Открытую книгу — символ чтения.
2. Закладку — символ маршрута и продолжения.
3. Тонкую сетку точек/линий — символ ИИ и связности фондов.

Текстовая часть: «ЧитАИ» в шрифте PT Serif, заглавная «Ч» с лёгкой акцентной диагональю, как будто закладка проходит через букву.

## Варианты

### Лого-knack (основной, горизонтальный)

```
┌──────────┐
│ [Знак]   │  Чи т АИ
│          │
└──────────┘
```

- Слева — графический знак (книга + закладка), 1×1 от высоты текста.
- Справа — текст «ЧитАИ» (PT Serif Bold, размер равен высоте знака).

### Лого вертикальный

```
[Знак]
ЧитАИ
```

- Знак сверху, текст снизу. Используется в квадратных аватарах.

### Только знак

Используется в favicon, в аватаре бота, в watermark на иллюстрациях.

### Только текст

«ЧитАИ» — для подвала, юр. бланков, узких контейнеров.

## Цветовые версии

1. Основная: знак — глубокий синий `#0E2A47`, закладка — терракотовый `#B0573B`, текст — глубокий синий.
2. Монохромная чёрная: всё `#1F2937`.
3. Монохромная белая: всё `#FFFFFF` (для тёмного фона).
4. Инверсия: знак белый, фон тёмно-синий.

## Минимальный размер

- В пикселях: 96 px по большей стороне.
- В мм: 20 мм по большей стороне.
- При меньших размерах — использовать только знак без текста.

## Охранная зона

- Свободная зона вокруг лого: не менее высоты буквы «Ч» по всем сторонам.
- Внутри охранной зоны не размещаются другие графические элементы.

## Запрещено

- Менять цвета на не предусмотренные палитрой.
- Поворачивать лого.
- Растягивать или сжимать (только пропорциональное масштабирование).
- Накладывать тени, обводки, эффекты.
- Помещать лого на пёстрые фотографии без подложки.
- Использовать эмодзи рядом с лого.
