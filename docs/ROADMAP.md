## 3. Конечное состояние «полностью готового продукта»

### 3.1 Что значит «готов к подаче куда угодно»

«Куда угодно» в твоём контексте = одновременно:
- **Нейрофест 2026** (акселератор) — нужны: код + видео-демо + питчдек + пилот + LOI
- **«Моя страна — Моя Россия»** (президентский конкурс) — нужны: видео + пилот + сильное B2G-обоснование
- **ФСИ, ИРИ, ПФКИ, РФРИТ** (гранты) — нужны: смета + ТЗ + команда + R&D-обоснование + предварительные LOI
- **Минкульт / Минпросвещения** (B2G-контракты) — нужны: пилот + юр.лицо + согласие с этическим протоколом + интеграция с гос. сервисами
- **Инвестор / акселератор** — нужны: traction (DAU, retention) + юнит-экономика + команда
- **App Store / RuStore** (если делаем PWA + натив) — нужны: production-инстанс + политика приватности + согласия

Все эти 6 направлений требуют **одного и того же** базового набора:

1. **Living URL** — `https://chitai.<домен>` доступен 24/7
2. **Working code** — все 3 mode + игры + боты + auth + monitoring
3. **Compliance** — 152-ФЗ + авторское право + этический протокол + Роскомнадзор уведомление + юр.лицо
4. **Traction** — ≥1 пилот, ≥3 LOI, ≥50 реальных пользователей
5. **Materials** — видео 90 сек, питчдек 12 слайдов, лендинг, сметы под грант
6. **Quality bar** — CI зелёный, тесты ≥80% coverage, бенчмарк MRR ≥0.85, бенч RAG публичный, security-аудит

| **R4.** LOI не получены за 2 недели | Высокая | Очень высокий | Параллельно подавать в 5–10 организаций. Бизнес-аналог: «cold outreach с 10% conversion = 1 LOI» |
| **R5.** Пилот не успеет за 30 дней | Высокая | Очень высокий | Запустить **сразу после первого LOI** (даже без полного функционала, MVP-режим). Лучше 14-дневный пилот, чем 0 |
| **R6.** Production-деплой падает | Низкая | Высокий | Terraform + auto-restart + Yandex Monitoring алёрты. Откат через `git revert` + `terraform apply` |
| **R7.** XSS в user uploads | Средняя | Очень высокий | DOMPurify (B1) + sanitize на бэке + content-security-policy header |
| **R8.** Стоимость LLM-вызовов взрывается | Средняя | Высокий | Rate-limit (B2) + idempotency + кэш + cost-monitoring в `dashboard.py` |
| **R9.** Авторское право нарушено (в Mode «Свой материал» пользователи могут загружать non-public-domain) | Средняя | Высокий | В Terms of Use: «вы загружаете только материалы, на которые имеете право». Audit-log + удаление по запросу. **Дисклеймер на upload** |
| **R10.** Минкульт отвергает после pivot («слишком похоже на Зачёт») | Средняя | Средний | POSITIONING.md A8: продаём Mode «Литература» + Pushkin Card как ключевое, Mode «Свой материал» — как «расширение для самостоятельной учёбы» |
| **R11.** Гранты ИРИ/ПФКИ — отказ | Высокая | Средний | Подавать одновременно во все 4 (ФСИ + ИРИ + ПФКИ + РФРИТ). Шанс получить хотя бы один ≥40% |
