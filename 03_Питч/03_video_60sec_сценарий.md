# Сценарий видеоролика ЧитАИ — 60 секунд

Формат: вертикальное и горизонтальное видео для соцсетей и e-mail. Звук обязателен. Субтитры обязательны.

## Раскадровка

| Сек | Кадр | Голос/субтитры | Бренд |

## Голос диктора (мужчина и женщина, по выбору)

- Тембр: спокойный, без излишней пафосности.
- Скорость: 110 слов/мин.
- Без излишней театральности.

## Музыкальное сопровождение

- Стиль: камерная фортепианная музыка или струнный квартет, спокойный.
- Громкость: -18 dB под голосом.
- Источник: Mubert PRO (российский) или собственный композитор.

## Субтитры

- Шрифт: PT Sans или Inter, 24 pt.
- Цвет: белый на полупрозрачной подложке.
- Включены по умолчанию.

## Технические требования

- Разрешение: 1920×1080 (горизонталь), 1080×1920 (вертикаль).
- FPS: 25.
- Кодек: H.264, аудио AAC 192 kbps.
- Длительность: ровно 60 секунд.
- Формат файла: MP4.
- Размер: ≤ 50 МБ для рассылки.

## Производство

- Сценарий: внутри команды.
- Голос диктора: профессиональный диктор (Yandex SpeechKit как резерв для черновика).
- Кадры: имитация скриншотов (внутренние мокапы) + 3–5 фото из стоков с российской лицензией (если стоки) или собственная съёмка.
- Музыка: лицензия Mubert PRO.

## Юридический чек-лист

- [ ] Все фото и музыка лицензированы.
- [ ] Логотипы YandexGPT, GigaChat, Yandex Cloud упомянуты в стилизованном виде, не нарушают гайдлайны брендов (или удалены, если есть сомнения).
- [ ] Никакие пользовательские данные в скриншотах не реальные (только мокданные).
- [ ] Согласие на участие у всех актёров (если съёмка с людьми).

## Версия для соцсетей

- 30 секунд: убрать кадры 26–34 и 42–50, ужать 18–26.
- 15 секунд: только сцена с маршрутом + лого + сайт.

## Чек-лист перед публикацией

- [ ] Видео хронометражем ровно 60 секунд.
- [ ] Текст диктора и субтитры синхронизированы.
- [ ] Голос диктора утверждён.
- [ ] Музыка лицензирована.
- [ ] Контакты и сайт — актуальны.
- [ ] Файл размером ≤ 50 МБ, формат MP4.
