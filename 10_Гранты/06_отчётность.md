# Регламент грантовой отчётности ЧитАИ

## 1. Общие положения

ЧитАИ обязуется предоставлять отчётность по грантам в соответствии с условиями каждого договора. Внутренний регламент определяет правила сбора, проверки, оформления и подачи отчётов.

## 2. Виды отчётности

### 2.1. Финансовая

- Авансовый отчёт (после получения транша).
- Промежуточный отчёт (раз в квартал или раз в полугодие в зависимости от условий).
- Заключительный отчёт.

### 2.2. Нефинансовая (программная)

- Описание выполненных работ по этапу.
- Достигнутые KPI.
- Аналитика по показателям.
- Кейсы пилотов.
- Презентации, публикации, методические материалы.

### 2.3. Аудиторская

- Аудиторское заключение по итогам проекта (если требуется).
- Привлечение внешнего аудитора.

## 3. Ответственные

| Зона | Ответственный |
|---|---|
| Финансовая отчётность | Бухгалтер (внешний контракт) |
| Программная отчётность | Менеджер проекта (CEO или ответственный по гранту) |
| Подача в систему грантодателя | Менеджер проекта |
| Координация с экспертами | Куратор контента / CTO |
| Подписание | CEO |

## 4. График отчётности

Зависит от условий гранта. Типичные графики:

- ФСИ «Старт»: ежеквартально + заключительный.
- ПФКИ: каждые 6 месяцев + заключительный.
- Сколково: раз в полугодие + заключительный.
- Минцифры: ежеквартально + годовой.
- Президентский фонд: каждые 6 месяцев.

Внутренние сроки закладываются за 7 дней до сроков по договору.

## 5. Форматы отчётности

### 5.1. Финансовый отчёт

- Списание расходов по статьям.
- Подтверждающие документы (платёжки, договора, акты).
- Сверка с бюджетом.
- Объяснение отклонений (если есть).

### 5.2. Программный отчёт

- Достижение KPI (план / факт).
- Описание выполненных работ.
- Доказательства (скриншоты, аналитика, кейсы).
- Список партнёров, привлечённых на этапе.

### 5.3. Презентация

- Для очного доклада на комиссии (если требуется).
- 10–15 слайдов.
- Цифры, факты, кейсы.

## 6. Принципы прозрачности

- Не выдумывать показатели.
- Все KPI должны быть подтверждены документально (логи системы, отчёты партнёров, аналитика).
- Расхождения с планом признаются и объясняются.
- При недостижении KPI — план корректирующих мер.

## 7. Аудит и проверка

### Внутренние

- Раз в квартал — внутренний аудит финансов и KPI.
- Раз в полугодие — внутренний аудит соответствия требованиям грантодателя.

### Внешние

- По запросу грантодателя — открытость для проверок.
- По итогам проекта — аудиторское заключение (если предусмотрено).
- Раз в год — независимый аудитор для всех грантов.

## 8. Хранение документов

- 5 лет минимум (требование законодательства).
- 10 лет для ключевых проектов.
- Электронный архив + бумажный (обязательные документы).

## 9. Защита от типичных ошибок

| Ошибка | Меры |
|---|---|
| Использование средств не по назначению | Раздельный счёт, согласование расходов с CFO |
| Несвоевременная отчётность | Календарь дедлайнов, дублирование контроля |
| Расхождение факта и плана | Регулярный пересмотр KPI, корректирующий план |
| Утрата подтверждающих документов | Электронный архив, бэкапы |
| Налоговые вопросы | Внешний бухгалтер, согласование с налоговой |

## 10. Корректировка показателей

Если планируемые KPI оказываются недостижимыми или превышают план, ЧитАИ:

1. Уведомляет грантодателя в течение 14 дней.
2. Предоставляет план корректировки (увеличение/уменьшение).
3. Согласовывает с грантодателем формальное изменение.
4. Не подаёт ложные данные.

## 11. Отчётность для общественности

После завершения проектов с грантом — публичная публикация отчёта:

- На сайте ЧитАИ (chitai.ru/about/grants).
- В блоге.
- В социальных сетях.
- В виде статьи в профильном СМИ.

Цель: открытая демонстрация социального эффекта, в т.ч. для будущих грантов.

## 12. Конфиденциальность

- Условия отдельных пилотов и партнёров — конфиденциально (если NDA).
- Обезличенные кейсы и метрики — публично.
- Согласие партнёров на публикацию — обязательно.

## 13. Чек-лист подачи отчёта

- [ ] Отчёт оформлен в соответствии с шаблоном грантодателя.
- [ ] Все KPI подтверждены документами.
- [ ] Финансовая часть согласована с бухгалтером.
- [ ] Программная часть согласована с CTO.
- [ ] Презентационная часть согласована с PR.
- [ ] Подписан CEO.
- [ ] Загружен в систему грантодателя за 5+ дней до дедлайна.
- [ ] Получено подтверждение приёма.
- [ ] Внутренняя копия в архиве.
