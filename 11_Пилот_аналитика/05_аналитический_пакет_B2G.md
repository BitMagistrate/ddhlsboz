# Аналитический пакет для B2G (региональные министерства)

## 1. Назначение

Документ описывает, какие данные ЧитАИ предоставляет региональному министерству культуры или образования в рамках B2G-договора. Пакет является ценностным предложением и юридически закреплённой частью услуги.

## 2. Принципы

- Все данные обезличены (152-ФЗ).
- Нет возможности идентифицировать конкретного пользователя.
- Доступ только согласованным сотрудникам министерства.
- Журнал доступа.

## 3. Состав пакета

### 3.1. Дашборд региона (онлайн)

Доступ через веб-интерфейс ЧитАИ, кабинет регионального министерства. Метрики обновляются раз в час.

#### Активность

- Активные пользователи (DAU, MAU) в регионе.
- Динамика по месяцам.
- Возрастная структура.
- Гендерная структура (с согласия).
- Город / район.

#### Учреждения культуры

- Подключено: библиотек, музеев, школ.
- Запросов через виджеты в день.
- Конверсия маршрут → визит.
- Топ-10 учреждений по активности.

#### «Пушкинская карта»

- Транзакций по карте в регионе через ЧитАИ-маршруты.
- Динамика по неделям.
- Топ-10 событий по доле от ЧитАИ-маршрутов.
- Доля по «Пушкинской» в общем потоке.

#### Тематические интересы

- Топ-30 тем по запросам в регионе.
- Динамика по сезонам.
- Возрастная структура каждой темы.
- Региональные особенности (например, какие региональные авторы / темы интересны).

#### ОГЭ/ЕГЭ

- Учеников в тренажёре в регионе.
- Средний балл по предметам (литература, история).
- Сравнение с прошлым годом (если данные есть).
- Топ-10 школ по активности.

### 3.2. Аналитический отчёт (раз в квартал)

Формат: PDF на 30–40 страниц.

#### Структура

1. Резюме (1 страница).
2. Краткие итоги квартала.
3. Активность.
4. Тематические интересы и тренды.
5. Анализ «Пушкинской карты» в регионе.
6. Учреждения культуры — рейтинг и динамика.
7. Рекомендации для региональной программы.
8. Сравнение с другими регионами (обезличенно).
9. Заключение и план на следующий квартал.

#### Подготовка

- Автоматизированная сборка из данных платформы.
- Ручная корректировка аналитиком.
- Согласование с партнёром.

### 3.3. Тематические исследования (по запросу)

Раз в квартал министерство может запросить тематический разрез:

- «Литература народов России» в регионе.
- «Военная проза» в регионе.
- «Серебряный век» в регионе.
- «Краеведение через классику» в регионе.
- Сравнение с другими регионами.

ЧитАИ готовит отчёт за 14 рабочих дней.

### 3.4. API для интеграции

- API для получения агрегированных данных.
- Авторизация через OAuth 2.0.
- Лимит: 1 000 запросов / час.
- Документация: api.chitai.ru/docs.

### 3.5. Соответствие нацпроекту

- Дашборд показателей нацпроекта «Культура» в регионе.
- Сравнение с целевыми значениями нацпроекта.
- Динамика по годам.

## 4. Защита данных

- Все ПД обезличены.
- Доступ только согласованным сотрудникам.
- Журнал доступа сохраняется 12 месяцев.
- Аудит — по запросу.
- Соответствие 152-ФЗ и Указу №250.

## 5. Использование данных министерством

Министерство имеет право:

- Использовать данные для региональных программ.
- Публиковать обезличенные результаты в отчётах.
- Передавать в федеральные органы (Минкульт РФ, Минобр РФ).

Министерство не имеет права:

- Передавать данные третьим коммерческим сторонам без согласования с ЧитАИ.
- Публиковать данные в формате, позволяющем идентификацию пользователей.
- Использовать данные в политических кампаниях.

## 6. Принципы публикации

При публикации данных министерством:

- Указание источника: «Данные предоставлены сервисом ЧитАИ, chitai.ru».
- Согласование текста с ЧитАИ за 7 дней.
- Возможность отказа ЧитАИ при некорректной интерпретации.

## 7. Ограничения

- Не предоставляются данные в режиме реального времени для отдельных пользователей.
- Не предоставляются персональные данные.
- Не предоставляются логи переписки с Куратором.
- Минимальный размер группы для агрегации: 10 пользователей.

## 8. Срок действия

- Доступ к дашборду — на весь срок договора.
- Отчёты — раз в квартал в течение срока договора.
- После прекращения договора — данные хранятся в архиве 1 год, затем удаляются.

## 9. Стоимость

Включена в стоимость B2G-договора (см. `09_B2G_B2B_продажи/07_тарифы.md`):

- Базовый региональный пакет: дашборд + ежеквартальный отчёт.
- Расширенный пакет: + тематические исследования (4 в год) + API.
- Полный пакет: + кастомизация дашборда + регулярные онлайн-консультации.

## 10. Поддержка

- Электронная почта: b2g@chitai.ru.
- Закреплённый аналитик ЧитАИ — 1 контакт на регион.
- Время ответа: 1 рабочий день.
- Очные встречи — по запросу, не реже 1 раза в полугодие.

## 11. Этический протокол B2G-аналитики

- Не использовать данные для рейтингования регионов в публичном пространстве (только для внутреннего планирования министерства).
- Не использовать данные для давления на учреждения.
- При обнаружении проблемы (например, низкий показатель в учреждении) — обращение в режиме «помощь и поддержка», не «контроль».
- Согласие на сравнение с другими регионами — добровольное.
