# ЧитАИ — пакет артефактов для акселератора Нейрофест

ЧитАИ — ИИ-куратор индивидуальных культурно-образовательных маршрутов по фондам Российской государственной библиотеки (РГБ) и Национальной электронной библиотеки (НЭБ). Стартап ориентирован на молодёжь 14–22 лет (через «Пушкинскую карту»), учителей литературы и истории, региональные библиотеки, музеи и подразделения Минкультуры/Минпросвещения.

Этот репозиторий — полная сборка артефактов проекта по структуре мастер-файла Нейрофеста: заявка, презентация, питч, юридический пакет, методички, прототип, маркетинг, B2G/B2B-продажи, гранты, пилот, FAQ.

## Технологический стек проекта

Полностью российский:

- LLM: YandexGPT 5 Pro, GigaChat MAX (фолбэк)
- Эмбеддинги: YandexGPT Embeddings, FRIDA / multilingual-e5
- Иллюстрации: Kandinsky 3.1
- TTS/ASR: Yandex SpeechKit
- БД: PostgreSQL 15 + pgvector, Redis для кэша сессий
- Облако: Yandex Cloud (Compute, Object Storage, MDB, KMS), резерв SberCloud
- Frontend: React 18 + TypeScript, Vite, Tailwind, shadcn/ui
- Backend: FastAPI 0.110, Python 3.11, Pydantic v2, aiogram 3 (Telegram-бот)
- Аналитика: Yandex Metrika + собственный сервис событий

## Принципы ответственного ИИ (см. `05_Юр_пакет/08_этический_протокол.md`)

1. Прозрачность: каждый ответ показывает источник и ссылку на фрагмент фонда.
2. Достоверность: при отсутствии источника модель отвечает «не знаю», без галлюцинаций.
3. Защита данных: соответствие 152-ФЗ, минимизация сбора, локализация в РФ.
4. Недискриминация: ровный сервис вне зависимости от региона, школы, дохода семьи.
5. Защита от вреда: фильтры на содержимое, возрастные режимы 14+ и 18+.
6. Открытый стек: документированные модели, версии промптов, журнал инцидентов.

## Как пользоваться репозиторием

1. Открыть `00_Канон/` — единая правда о проекте.
2. Любой текст подаётся в комплекте: основной файл + источник из `00_Канон/`.
3. Перед отправкой партнёру убедиться, что цифры в заявке, презентации и питче совпадают.
4. Юридические шаблоны в `05_Юр_пакет/` обязательно проверяет юрист до подписания.
5. Прототип в `07_Прототип/` и `backend/` + `frontend/` — это демо для жюри, а не продакшн.

## Отказ от ответственности

Юридические шаблоны (Политика ПД, согласия, договор, NDA, реестр ПД) — рабочие черновики. Перед публикацией и заключением сделок их обязан проверить квалифицированный юрист. Партнёрства, обозначенные как «в стадии переговоров», не имеют статуса подписанного соглашения до получения соответствующего письма поддержки или контракта.
